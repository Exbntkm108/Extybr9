Описание

"Random Password Generator" — это GUI-приложение, разработанное на Python с использованием tkinter. Оно позволяет пользователям генерировать случайные пароли заданной длины, выбирая, включать ли в пароль цифры, буквы и/или спецсимволы. Приложение также сохраняет историю всех сгенерированных паролей в файл password_history.json и отображает ее в таблице. Для управления версиями проекта настроен локальный Git-репозиторий.

Примеры использования


    Запуск:


        Сохраните код как password_generator.py.

        Убедитесь, что установлен Python.

        Откройте терминал, перейдите в директорию с файлом и запустите:
        bash
        python password_generator.py

        При первом запуске будет создан .git репозиторий и файл .gitignore.

    Генерация пароля:


        Настройте длину пароля с помощью ползунка (от 4 до 128 символов).

        Отметьте галочками желаемые типы символов: "Цифры", "Буквы", "Спецсимволы". Минимум один тип должен быть выбран.

        Нажмите кнопку "Сгенерировать пароль".

        Сгенерированный пароль отобразится в поле "Пароль".

        Сгенерированный пароль будет добавлен в "Историю паролей".

    Просмотр истории:


        Ниже блока генерации находится таблица "История паролей".

        Она показывает время генерации, сам пароль, его длину и использованные типы символов. Новые записи добавляются вверху.

    Git:


        Скрипт автоматически инициализирует Git-репозиторий при первом запуске.

        Файл password_history.json добавлен в .gitignore, чтобы не сохранять пароли в системе контроля версий.

        Для выгрузки проекта на удаленный репозиторий (например, GitHub):
        bash
        git add .
        git commit -m "feat: Password generator with history and git"
        git branch -M main
        git remote add origin <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
        git push -u origin main
